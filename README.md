# react-native-owl [![github][github-image]][github-url]

> **Work In Progress**: Visual regression testing for React Native

## Installation

```sh
yarn add -D react-native-owl
# or
npm install -D react-native-owl
```

## CLI

### Building the app

```bash
owl build
```

### Running the tests

```bash
owl test
```
